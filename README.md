# rsync examples

* [rsync](https://man7.org/linux/man-pages/man1/rsync.1.html)

       Local:
 >          rsync [OPTION...] SRC... [DEST]

## options

> -u  update files
> -a  archive files
> -v  verbose
> -z  compress files during transfer
> -h  output numers in human readable format
> -n  dry run (no transfer)
